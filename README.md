# Frontend - Sitio Web de Noticias

Frontend de un sitio web de noticias desarrollado con React, Typescript, Tailwind.   
Este repositorio contiene la interfaz de usuario del proyecto y se encuentra en una etapa inicial de desarrollo.

---

## Estado del proyecto

 **En desarrollo**  
Actualmente el proyecto cuenta con:

- Creación del proyecto base
- Estructura inicial de carpetas
- Implementación del Navbar 
- Implementación de Home 
- Configuración inicial del entorno de desarrollo
- Configuración de variables de entorno y axios
- Cración e implementación del servicio get News.

---

## Tecnologías utilizadas

- **React**
- **Vite**
- **JavaScript**
- **CSS / Tailwind CSS V.3** 
- **Axios**
- **React-Query**
- **React-Router**
- **dayjs**
